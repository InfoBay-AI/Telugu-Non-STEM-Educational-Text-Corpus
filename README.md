# Telugu-Non-STEM-Educational-Text-Corpus
**Dataset Description:**

This dataset is a **large-scale collection of Telugu Non-STEM Textbook data**, designed to support the development of advanced NLP systems and AI models for educational understanding, comprehension, and language learning in Telugu.

Additionally, this dataset can be used in pipelines for **Supervised Fine-Tuning (SFT) and Reinforcement Learning with Human Feedback (RLHF) workflows**, improving model performance in multilingual comprehension, summarization, and reasoning tasks.

**Key Use Cases**

    -Text-based Question Answering (Descriptive & MCQ)
    -Named Entity Recognition (NER) in non-STEM content
    -Text summarization and comprehension
    -Automated tutoring and educational assistants
    -Knowledge retrieval systems for humanities subjects
    -Model evaluation and benchmarking

**Dataset Specification**

    -Modality: Telugu
    -Type: Educational / Non-STEM
    -Data Source: Curated academic textbooks and educational material
    -Data Nature: Real-world and curated data
    -Books: 2,672

**Value of This Dataset**

    -Enables learning of STEM subjects in Telugu
    -Enhances mathematical and logical reasoning capabilities in AI models
    -Supports multilingual scientific and technical NLP systems
    -Helps build AI-powered educational and tutoring platforms
    -Improves accuracy in problem-solving and analytical reasoning tasks
    -Strengthens LLM performance in structured and formula-based domains
